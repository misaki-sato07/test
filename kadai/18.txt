■問題文
指定されたテキストファイルを"リアルタイム"に読み込み、その内容を出力するプログラムを作成せよ。

■入力
対象のファイルはコマンドの引数で与えられる。
usage: command fileA

■出力
- コマンド実行時にファイルの中身を読み込み、全行標準出力する
- コマンドは外部からの終了命令を受けるまでファイルを監視し続ける
- 監視対象のファイルに何かが書き込まれた場合、その内容を読み込み追加で標準出力に出力する。

■制約
- 以下のようなシステムコールを使用すること
    - open
    - close
    - read
    - write
    - stat/fstat/lstat
- 以下は使わない
    - fopen/fcloseなど
    - inotify

■その他動作
- バイナリファイルは考慮しないでもよい。
- ファイルの末尾に必ず改行コードがある前提でよいものとする。
- ファイル内容の変更は、末尾に新規行としてデータが追加された場合のみ考慮すればよいものとする。
  以下を考慮する必要はない
    - ファイル途中へのデータ挿入
    - データの書き換え
    - データの削除
- 処理の途中で異常が発生した場合は、終了コード1で終了する
- 外部から終了命令を受けるかエラーが発生するまでコマンドは無限待受をする

■入力例
$ command fileA

■出力例
1. fileAの内容を標準出力する
2. fileAにabcを書き込むと、コマンドはabcを出力する。
3. 続けてfileAにdefを追加で書き込むと、コマンドはdefを出力する。

$ cat fileA
123

$ command fileA
123
abc                     | 別プロセス $ echo "abc" >> fileA
def                     | 別プロセス $ echo "def" >> fileA
